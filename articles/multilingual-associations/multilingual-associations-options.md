## Description

La page *Associations multilingues : Options* ne comporte qu'un onglet Permissions, qui est couvert comme un élément commun.

* [Barres d'outils](jdocmanual?article=help/common-elements/toolbars).
* [L'onglet Permissions](jdocmanual?article=help/common-elements/edit-permissions).

## Comment y accéder

* Sélectionnez **Composants → Associations multilingues** dans le menu d'administration.
* Sélectionnez le bouton **Options** dans la barre d'outils.
